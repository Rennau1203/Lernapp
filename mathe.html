Analysis & Kurvendiskussion

Ableitungen, Extrempunkte, Wendepunkte, vollständige Funktionsuntersuchung

Die 7 Schritte der Kurvendiskussion

Bei einer vollständigen Funktionsuntersuchung werden folgende Eigenschaften untersucht:

  • 1

    Definitionsbereich

    Für welche x-Werte ist die Funktion definiert? Bei ganzrationalen Funktionen immer D = ℝ.

  • 2

    Symmetrie

    Achsensymmetrie: f(–x) = f(x) · Punktsymmetrie: f(–x) = –f(x)

  • 3

    Verhalten für x → ±∞

    Höchste Potenz bestimmt das Grenzverhalten. Vorzeichen & Grad beachten.

  • 4

    Achsenschnittpunkte

    y-Achse: f(0) einsetzen. x-Achse (Nullstellen): f(x) = 0 lösen.

  • 5

    Extrempunkte

    f'(x) = 0 (notwendige Bedingung), dann f''-Kriterium oder VZW-Kriterium.

  • 6

    Wendepunkte

    f''(x) = 0 (notwendige Bedingung), dann f'''-Kriterium oder VZW-Kriterium von f''.

  • 7

    Graph

    Alle berechneten Punkte einzeichnen und verbinden.

Durchgerechnetes Beispiel

f(x) = –x³ + 12x² – 21x – 34 = –(x+1)(x²–13x+34)

D = ℝ
f(–x) = x³ + 12x² + 21x – 34 ≠ f(x) → keine Achsensymmetrie
–f(–x) = –x³ – 12x² – 21x + 34 ≠ f(x) → keine Punktsymmetrie

Höchste Potenz: –x³

lim(x→+∞) f(x) = –∞
lim(x→–∞) f(x) = +∞
y-Achse
f(0) = –34 → P(0|–34)
Nullstellen
0 = –(x+1)(x²–13x+34) x₁ = –1 → N₁(–1|0) x₂,₃ = 13/2 ± √((13/2)²–34) ≈ 9,37 und 3,63
f'(x) = –3x² + 24x – 21 f''(x) = –6x + 24

Notwendige Bedingung: f'(x) = 0

–3x² + 24x – 21 = 0 → x² – 8x + 7 = 0 x₁ = 1, x₂ = 7
Hinreichende Bedingung (f'')
f''(1) = –6·1+24 = 18 > 0 → Tiefpunkt T(1|–34) f''(7) = –6·7+24 = –18 < 0 → Hochpunkt H(7|64)
f''(x) = –6x + 24 = 0 → x = 4 f'''(x) = –6 ≠ 0 → Wendepunkt bestätigt
f(4) = –64 + 192 – 84 – 34 = 10 → W(4|10) f'''(4) = –6 < 0 → Links-Rechts-Wendepunkt (LR-W)

Symmetrie – Kurzregel

Achsensymmetrie (y-Achse)

f(–x) = f(x)
Nur gerade Potenzen → alle Exponenten gerade

Punktsymmetrie (Ursprung)

f(–x) = –f(x)
Nur ungerade Potenzen → alle Exponenten ungerade

Ableitungsregeln im Überblick

f(x) = xⁿ → f'(x) = n · xⁿ⁻¹

Beispiele:

f(x) = x⁴ → f'(x) = 4x³ → f''(x) = 12x² → f'''(x) = 24x
f(x) = 4/x = 4x⁻¹ → f'(x) = –4x⁻² → f''(x) = 8x⁻³
[f(x) + g(x)]' = f'(x) + g'(x)

Jeder Summand wird einzeln abgeleitet.

[c · f(x)]' = c · f'(x)

Konstante Faktoren bleiben beim Ableiten erhalten.

Man kann eine Funktion mehrfach ableiten:

f(x) = x⁴ – 8x³ + x f'(x) = 4x³ – 24x² + 1 f''(x) = 12x² – 48x f'''(x)= 24x – 48

Wichtig: Die dritte Ableitung f''' wird bei der Wendepunkt-Überprüfung (hinreichende Bedingung) verwendet.

Durchschnittliche vs. momentane Änderungsrate

Durchschnittliche Änderungsrate

Steigung der Sekante zwischen zwei Punkten:

m = (f(x₂)–f(x₁)) / (x₂–x₁)
Momentane Änderungsrate

Steigung der Tangente = Wert der Ableitung:

v(t) = f'(t)
Beispiel – f(x) = x²
Durchschnittl. Änderungsrate [0; 10]
P₁(0|0), P₂(10|100) → m = (100–0)/(10–0) = 10 m/s
Momentane Änderungsrate bei t = 10
f'(x) = 2x → f'(10) = 20 m/s

Notwendige Bedingung

Wenn bei xE ein lokales Extremum von f liegt, dann gilt: f'(xE) = 0

Umkehrung gilt nicht! → f'(xE) = 0 bedeutet nur potentielles Extremum (könnte auch Sattelpunkt sein).

Hinreichende Bedingung – zwei Methoden

Hochpunkt

f'(xE) = 0 und f''(xE) < 0
(linksgekrümmt = Rechtskrümmung zur Tangente)

Tiefpunkt

f'(xE) = 0 und f''(xE) > 0
(rechtsgekrümmt)

Sattelpunkt

f'(xE) = 0 und f''(xE) = 0
→ keine Entscheidung möglich, VZW-Kriterium nötig!

Teststellen links und rechts von xE in f' einsetzen:

Hochpunkt

f' wechselt von + nach –

Tiefpunkt

f' wechselt von – nach +

Sattelpunkt

f' hat keinen Vorzeichenwechsel

Beispiel Sattelpunkt – f(x) = x³
f'(x) = 3x² = 0 → x = 0 f''(0) = 0 → keine Entscheidung Teststellen: f'(–1) = 3 > 0, f'(1) = 3 > 0 → kein VZW → Sattelpunkt bei (0|0)

Vorgehen Schritt für Schritt

  • 1

    f' und f'' berechnen

    Beide Ableitungen aufstellen.

  • 2

    Notwendige Bedingung

    f'(x) = 0 lösen → potentielle Extremstellen xE

  • 3

    Hinreichende Bedingung

    f''(xE) einsetzen. Falls = 0: VZW-Kriterium anwenden.

  • 4

    y-Koordinate berechnen

    f(xE) einsetzen → Punkt angeben.

Notwendige Bedingung

Wenn bei xW ein Wendepunkt von f liegt, dann gilt: f''(xW) = 0

Links-Rechts-Wendepunkt (LR-W)

f zuerst linksgekrümmt, dann rechtsgekrümmt
f'' wechselt von + nach –
f' hat ein Maximum bei xW

Rechts-Links-Wendepunkt (RL-W)

f zuerst rechtsgekrümmt, dann linksgekrümmt
f'' wechselt von – nach +
f' hat ein Minimum bei xW

Hinreichende Bedingung – zwei Methoden

LR-Wendepunkt

f''(xW) = 0 und f'''(xW) < 0

RL-Wendepunkt

f''(xW) = 0 und f'''(xW) > 0

f''(x) links und rechts von xW auswerten:

LR-Wendepunkt

f'' wechselt von + nach –

RL-Wendepunkt

f'' wechselt von – nach +

Durchgerechnetes Beispiel

f(x) = ½x³ – 3/2 x²

Schritt für Schritt
Ableitungen
f'(x) = 1,5x² – 3x f''(x) = 3x – 3 f'''(x)= 3
Notwendige Bedingung: f''(x) = 0
3x – 3 = 0 → x = 1
Hinreichende Bedingung
f'''(1) = 3 > 0 → Rechts-Links-Wendepunkt (RL-W) f(1) = ½–3/2 = –1 → W(1|–1)
Weiteres Beispiel – f(x) = 1/30 x⁴ – 4/5 x²
Ableitungen
f'(x) = 2/15 x³ – 1,6x f''(x) = 0,4x² – 1,6 f'''(x)= 0,8x
Notwendige Bedingung
0,4x² – 1,6 = 0 → x² = 4 → x = ±2
Hinreichende Bedingung
f'''(2) = 0,8·2 = 1,6 > 0 → RL-W → W(2|–8/5) f'''(–2) = 0,8·(–2) = –1,6 < 0 → LR-W → W(–2|–8/5)

f und f'' einander zuordnen

Schlüsselzusammenhänge, die du kennen musst:

  • An einem Wendepunkt von f hat f'' eine Nullstelle.
  • Nullstelle von f'' und Wendepunkt von f haben den gleichen x-Wert.
  • Wenn f'' die x-Achse von + nach – kreuzt → LR-Wendepunkt bei f.
  • Wenn f'' die x-Achse von – nach + kreuzt → RL-Wendepunkt bei f.
  • Hochpunkt von f → Nullstelle von f' mit VZW + nach –
  • Tiefpunkt von f → Nullstelle von f' mit VZW – nach +
  • Extremstelle von f' → Wendepunkt von f
  • Nullstelle von f'' → Wendepunkt von f
f'' > 0

f ist rechtsgekrümmt (Linkskrümmung = konkav)

f'' < 0

f ist linksgekrümmt (Rechtskrümmung = konvex)

Klausur-Zuordnungsaufgaben

Laut Mitschrift: Graphen I, II, III (f) werden den Graphen A, B, C (f'') zugeordnet.

Graph I → f'' = B (Gerade)

Graph I ist ein Polynom 3. Grades. Zunächst linksgekrümmt, dann rechtsgekrümmt (LR-W). f'' muss VZW von + nach – haben → das trifft auf die Gerade B zu. Nullstelle von f'' = Wendepunkt von f haben denselben x-Wert.

Graph II → f'' = C (Parabel)

Graph II ist Polynom 4. Grades: erst RL-W, dann LR-W. f'' hat VZW von – nach + und danach + nach –. Das trifft auf die Parabel C zu. Nullstellen von f'' = Wendepunkte von f teilen x-Wert.

Graph III → f'' = A (neg. Parabel)

Graph III ist eine nach unten geöffnete Parabel. Über den gesamten Bereich rechtsgekrümmt → f'' muss überall negativ und konstant sein. Das trifft auf die negative Parabel A (unterhalb x-Achse) zu.

Wie skizziert man f aus f' ?

  • 1

    Nullstellen von f' finden

    Dort hat f Extrempunkte. VZW + nach – = Hochpunkt. VZW – nach + = Tiefpunkt.

  • 2

    Extrema von f' finden

    Dort hat f Wendepunkte.

  • 3

    Vorzeichen von f' beachten

    f' > 0 → f steigt. f' < 0 → f fällt.

  • 4

    Unendlich viele Lösungen

    Da beim Ableiten Konstanten wegfallen, kann f beliebig in y-Richtung verschoben sein.
